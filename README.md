# Hospital Queue Management System

A full-stack web application for managing hospital patient queues, built with HTML, CSS, JavaScript, Node.js, and PostgreSQL.

## Features

- User authentication (login/register)
- Patient queue management (add/remove patients)
- Responsive frontend

## Setup

1. Clone the repository.
2. Install dependencies: `npm install`
3. Set up PostgreSQL database and run `schema.sql` to create tables.
4. Set environment variables: `DATABASE_URL` for PostgreSQL connection.
5. Run the server: `npm start`

## Deployment on Render

1. Push to GitHub.
2. Connect Render to your GitHub repo.
3. Set up PostgreSQL database on Render.
4. Configure environment variables in Render dashboard.
5. Deploy the web service.
